## 🌴 About Me

웹 프론트엔드 개발자 직무를 희망하여 부트캠프를 통하여 한 걸음 더 나아가고 있습니다.
프론트엔드의 개발에서 중요한 것은 나 혼자만 알아보는 코드를 작성하는 행동은 지양하는 것과 사용자의 편리함, 가독성, 접근성을 향상하고 만든 결과물이 사람들의 삶에 긍정적인 영향을 주게 하는 것에 우선을 두고 개발하는 것으로 생각합니다.
개발할 때 마음가짐은 어떻게 하면 내가 작성한 코드를 주변 동료도 쉽게 파악하고, 어떻게 하면 사용자의 접근성 및 가독성이 올라가며 성공적인 서비스를 만들어 갈 수 있을까? 를 중점에 두고 있습니다.
