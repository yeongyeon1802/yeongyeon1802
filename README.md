## 🌴 About Me

웹 프론트엔드 개발자 직무를 희망하여 부트캠프를 통하여 한 걸음 더 나아가고 있습니다.
<br />
프론트엔드의 개발에서 중요한 것은 나 혼자만 알아보는 코드를 작성하는 행동은 지양하는 것과
<br />
사용자의 편리함, 가독성, 접근성을 향상하고 만든 결과물이 사람들의 삶에
<br />
긍정적인 영향을 주게 하는 것에 우선을 두고 개발하는 것으로 생각합니다.
<br />
개발할 때 마음가짐은 어떻게 하면 내가 작성한 코드를 주변 동료도 쉽게 파악하고,
<br />
어떻게 하면 사용자의 접근성 및 가독성이 올라가며 성공적인 서비스를 만들어 갈 수 있을까? 를 중점에 두고 있습니다.

## 💪 Strength

### 클린 코드 작성

코드를 작성할 때 나만이 알아보고 수정하기 쉬운 코드가 아닌
<br />
시맨틱 태그 사용, 의미 있는 클래스 이름 정하기, 맥락파악이 힘들 것 같으면 주석도 붙이기 등등을 활용하여
<br />
동료도 쉽게 맥락 파악이 가능한 코드를 작성하는 것을 항상 염두에 두고 있습니다.
<br />
또한, 사용자 접근성, 가독성 향상 및 SEO(검색 엔진 최적화)를 위해서도
<br />
무분별한 &lt;div&gt;태그를 사용하는 것 대신 적절한 시맨틱 태그 활용은 필수라고 생각하기에
<br />
이 부분을 항상 중점에 두고 코드를 작성하고 있습니다.

### 웹 디자이너와 원활한 의사소통 가능

웹 디자이너들이 사용하는 기본 툴들을 이해하고 다룰 줄 아는 것뿐만 아니라
<br />
웹 UI/UX 수업 과정을 통해서 디자이너가 어떤 의도를 가지고 이런 디자인을 했는지
<br />
또는 디자이너의 요구사항은 어떻게 되는지 등등을 이해할 수 있으며,
<br />
이 때문에 디자이너들과 원활한 협업 및 의사소통을 할 수 있습니다.
